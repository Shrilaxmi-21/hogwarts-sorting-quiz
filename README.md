A Harry Potter-inspired web app that sorts you into your Hogwarts house — Gryffindor, Slytherin, Ravenclaw, or Hufflepuff — based on your answers to 8 personality questions.

## 🛠️ Built With
- HTML
- CSS
- JavaScript (Vanilla)

## ✨ Features
- 8 personality-based sorting questions
- Score tracking for each Hogwarts house
- Animated result reveal
- Hogwarts-themed dark UI with gold accents
- Play Again option

## 🚀 How to Run
1. Clone the repository
2. Open `index.html` in your browser
3. Click "Get sort now" to begin the sorting ceremony
